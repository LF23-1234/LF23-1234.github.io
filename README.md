How to publish:
1. Create a GitHub repository and push these files to `main`.
2. In repo Settings → Pages, select branch `main` and folder `/` (root).
3. Save and wait for the site URL to appear.

Notes:
- Theme preference is persisted in localStorage.
- Replace placeholder images (picsum.photos) with your assets before final submission.
